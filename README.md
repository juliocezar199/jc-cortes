# JC Cortes

Sistema de controle de produção e financeiro da JC Cortes.

## Stack do projeto

- Next.js
- TypeScript
- Supabase

## Funcionalidades

- Cadastro de clientes
- Criação de pedidos / mapas
- Controle de modelos e pares
- Baixas de corte
- Controle financeiro
- Dashboard com métricas

## Execução em desenvolvimento

```bash
npm run dev
